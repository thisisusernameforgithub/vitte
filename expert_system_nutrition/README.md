### 1. Инструкция по развертыванию

> Ubuntu 22.04, 24.04 | Astra Linux 1.7.x, 1.8.x

#### 1. Клонирование репозитория, выгрузка датасета

```bash

git clone https://github.com/thisisusernameforgithub/vitte.git && cd vitte/expert_system_nutrition
```

#### 2. Создание и активация виртуального окружения

```bash

python3 -m venv venv && source venv/bin/activate
```


### 2. Руководство пользователя

#### 1. Регистрация и вход

- Регистрация:
  1. откройте главную страницу приложения в браузере
  2. нажмите на ссылку "Регистрация" в правом верхнем углу
  3. заполните форму: введите логин пользователя и пароль
  4. нажмите кнопку "Зарегистрироваться"
- Вход:
  1. после регистрации или при повторном посещении сайта используйте форму "Вход" на главной странице
  2. введите логин и пароль
  3. нажмите кнопку "Войти"

#### 2. Основной интерфейс

После входа в систему вы увидите главный экран, который содержит:

- Поле поиска продуктов - для быстрого нахождения нужных продуктов питания
- Таблица "Рацион на сегодня" - здесь отображаются все продукты, добавленные в рацион, с указанием их веса и пищевой ценности
- Итоговая таблица - показывает суммарное количество калорий, белков, жиров и углеводов
- Кнопка "Получить оценку рациона" - запускает анализ рациона

#### 3. Ведение рациона

1. Найдите продукт. Начните вводить название продукта в поле "Найдите продукт". Под полем появится всплывающее меню с списком продуктов, соответствующих вашему запросу (наименования продуктов на английском языке (!))
2. Добавьте продукт:
   - нажмите на интересующий вас продукт в списке
   - во всплывающем окне введите вес продукта в граммах
   - нажмите кнопку "Добавить"
   - продукт будет добавлен в таблицу "Рацион на сегодня", тем временем итоговые значения питательных веществ обновятся
   - повторите этот процесс для всех приемов пищи в течение дня

#### 4. Получение оценки рациона

1. После того как вы добавили все продукты за день, нажмите на кнопку "Получить оценку рациона"
2. Система проанализирует итоговое соотношение белков, жиров и углеводов с помощью модели машинного обучения
3. На экране появится результат оценки - класс, к которому был отнесен ваш рацион (например, "Сбалансированный", "Недостаток белка", "Избыток жиров" и т.д.). Эта оценка поможет вам понять, насколько ваш рацион соответствует принципам здорового питания

#### 5. Выход из системы

Чтобы завершить сеанс работы с приложением, нажмите на кнопку "Выход" в правом верхнем углу экрана
